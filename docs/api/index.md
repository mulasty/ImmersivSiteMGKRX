# API Reference

The project exposes one active public JavaScript module and one legacy side-effect module.

| Module | Purpose |
| --- | --- |
| [`src/app.js`](app.md) | Active vanilla JavaScript controller for the Tailwind/Vite page. |
| [`src/main.js`](main.md) | Legacy Three.js/WebGL flight scene retained for reference. |

Examples in these pages are written as ES modules and are expected to run in either a browser or Node.js where noted.
