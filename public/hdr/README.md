# Cinematic HDRI

Place a 4K studio HDRI here:

```text
public/hdr/studio-4k.hdr
```

When this file exists, the app loads it automatically as the PBR environment map.
Without it, Three.js uses a lightweight procedural studio environment.

Good CC0 HDRI sources:

- Poly Haven HDRIs: https://polyhaven.com/hdris
